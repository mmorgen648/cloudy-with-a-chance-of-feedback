headers = ["Host"]

Fix: API Routing über CloudFront

Initiales Problem:

API Requests über CloudFront funktionierten nicht zuverlässig.

Fehlerbilder:

502 Bad Gateway
503 Service Temporarily Unavailable

Root Cause:

CloudFront behandelte API Requests wie statische Inhalte
und leitete Requests nicht vollständig an den ALB weiter.

Fix:

Ordered Cache Behavior für:

/api/*

Konfiguration:

- Caching deaktiviert
- Alle HTTP Methoden erlaubt
- Cookies werden weitergeleitet
- Query Strings werden weitergeleitet
- Host Header wird weitergeleitet

Ergebnis:

an den ALB und anschließend an das Backend
