# CloudyWithAChanceOfFeedback

Cloud-native Feedback Analyzer built on AWS using Java Spring Boot, Kubernetes (EKS) and Terraform.

## Project Goal

Analyze customer text feedback using AWS Comprehend (Sentiment + Key Phrases) and display results in a simple web dashboard.

## Tech Stack

- Java Spring Boot (Backend)
- HTML / CSS / Bootstrap (Frontend)
- AWS EKS (Kubernetes)
- AWS RDS PostgreSQL
- AWS Cognito (Authentication)
- AWS Comprehend (ML)
- Terraform (Infrastructure as Code)
- GitHub Actions (CI/CD)
