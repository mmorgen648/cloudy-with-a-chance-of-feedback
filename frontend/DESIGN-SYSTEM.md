# DESIGN SYSTEM

Projekt: Cloudy With A Chance Of Feedback

---

## 1. Grundidee

Ziel ist ein professionelles SaaS-UI mit:

- Dunklem Grundlayout
- Violett (Primary) als Haupt-Akzentfarbe
- Smaragd (Emerald) für positive Zustände
- Ruhige, moderne Typografie
- Keine verspielten Effekte
- Kein Cartoon-Style

---

## 2. Farbkonzept

### Background

- Main Background: #0f172a
- Card Background: #1e293b
- Input Background: #0b1220

### Text

- Primary Text: #e2e8f0
- Muted Text: #94a3b8

### Primary (Akzentfarbe)

Verwendung:

- Buttons
- Hover-Zustände
- Wichtige Interaktionen
- Titel-Akzent

Farben:

- Primary: #7c3aed
- Hover: #6d28d9
- Soft Accent (Titel): #a78bfa

### Emerald (nur für Status / Sentiment)

Verwendung:

- Positive Sentiments
- Erfolgszustände
- Statistiken

Farben:

- Emerald: #10b981
- Soft Emerald: #34d399

Wichtig:
Emerald wird NICHT als Button-Primärfarbe verwendet.

---

## 3. Typografie

Font: Space Grotesk

Gewichte:

- 400 → Normaler Text
- 500 → Labels
- 600 → Titel

Keine zweite Schriftart.

Hierarchie:

- Page Title: 26–30px
- Section Title: 18–22px
- Body Text: 14–16px

---

## 4. Header-Regeln

- Contained Layout (max-width ~1100px)
- Logo links
- Titel zentriert
- Navigation rechts
- Kompakter Header
- Keine übermäßige vertikale Höhe

---

## 5. Cards

- Leichter Gradient
- Border subtil (#334155)
- Border-Radius: 12–16px
- Kein starker Glow
- Kein starker Schatten

---

## 6. Buttons

Primary Button:

- Violett
- Leichter Hover-Dunkelton
- Keine Animation
- Keine Shadow-Explosion

Sekundär Button:

- Transparent
- Border #334155
- Hover Background #1e293b

---

## 7. Admin Dashboard (später)

Sentiment-Farben:

- POSITIVE → Emerald
- NEGATIVE → Rot (noch festzulegen)
- NEUTRAL → Grau
- MIXED → Purple Accent

Statistik:

- Nur letzte 7 Tage
- Kein Live-Update
- Aktualisieren-Button

---

## 8. Prinzipien

- Kein Overdesign
- Kein visuelles Chaos
- Keine unruhigen Animationen
- Keine Farbexplosion
- Klarer SaaS-Look
